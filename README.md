# ColorLab — CMYK ↔ RGB ↔ HLS (PySide6)

Простое учебное приложение для исследования конверсий между CMYK, RGB и HLS с интерактивным UI.

## Требования
- Python 3.10+
- PySide6

## Запуск
```bash
python -m venv .venv
# Windows
. .venv\\Scripts\\activate
# Unix
# source .venv/bin/activate

pip install -r requirements.txt
python main.py
